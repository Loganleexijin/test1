位置：首页顶部导航栏（Header），位于“状态文字”旁边或下方。

形态：胶囊按钮 (Capsule Button)。

显示内容：当前计划图标 + 名称 (e.g., ⚡️ 16:8 计划 ▾)。

交互：点击后，从底部弹出一个 半屏卡片浮层 (Bottom Sheet)。

优势：用户随时可见当前目标，且切换成本低（不用跳转页面），符合“灵动”的理念。

2. 计划内容矩阵 (Content Strategy)
我们需要将枯燥的数字（16:8）包装成有吸引力的生活方式。我为你整理了 5 个标准档位，文案经过润色，强调收益而非忍耐。

🟢 Level 1: 生物钟韵律 (Circadian Rhythm)
配比：12:12 (断食 12h / 进食 12h)

适合人群：绝对新手、肠胃敏感者。

UI 标签：新手友好 极易坚持

核心文案：

口号：“睡一觉就完成了。”

收益：让消化系统在夜间彻底休息，改善晨起水肿。

🟡 Level 2: 黄金标准 (The Classic) —— 默认选项
配比：16:8 (断食 16h / 进食 8h)

适合人群：减脂主力军、上班族。

UI 标签：最热门 代谢平衡

核心文案：

口号：“跳过一顿早餐，换来全天的高效燃脂。”

收益：触达轻度燃脂期，有效控制体重，保持精力充沛。

🟠 Level 3: 战士模式 (Warrior Mode)
配比：18:6 (断食 18h / 进食 6h)

适合人群：有经验的断食者、平台期用户。

UI 标签：深度燃脂 进阶挑战

核心文案：

口号：“将进食压缩在午后，给身体更深层的净化。”

收益：稳定进入自噬初期，加速顽固脂肪分解。

🟣 Level 4: 僧侣修行 (Monk Fast / OMAD)
配比：23:1 (一天一餐 One Meal A Day)

适合人群：硬核玩家、极客。

UI 标签：高难度 细胞重塑

核心文案：

口号：“每日一次的盛宴，其余时间留给大脑和灵感。”

收益：深度细胞自噬，极致的胰岛素敏感性修复。

⚪️ Level 5: 自定义模式 (Custom Flow)
配比：用户手动拨盘。

场景：弹性生活（比如今天想断 14 小时，明天想断 20 小时）。

3. 弹窗 UI 布局设计 (Bottom Sheet UI)
当用户点击顶部的“16:8”胶囊时，弹出的浮层应包含以下元素：

顶部标题：

“选择你的节奏 (Choose Your Rhythm)”

横向滑动卡片 (Horizontal Scroll) 或 垂直列表：

建议垂直列表，信息展示更全。

左侧：可视化时钟图标（饼图，展示禁食区和进食区）。

中间：计划名称 + 简短收益（如上文）。

右侧：难度指示条（如：●●○○○）。

选中态：

当前选中的计划会有高亮边框和打钩图标。

智能预览 (Smart Preview)：

当用户点击某个计划（未确认）时，界面下方动态显示预测：

“如果选择 18:6，您今天的断食将在 14:00 结束。” —— 这个极其重要，帮用户算账。

底部按钮：

“确认切换” (Update Plan)。

4. 智能逻辑规则 (Smart Logic)
这是产品体验的“细节魔鬼”，必须处理好以下逻辑：

断食中切换 (Switching While Fasting)：

场景：用户正在进行 16:8，已经断了 14 小时，突然觉得自己行了，想改成 18:6。

逻辑：无缝延长。进度条目标从 16 变 18，圆环回退（百分比变小），但“已断食时长”不变。

反馈：提示用户“目标已更新，加油，还差 4 小时！”

缩短计划 (Downgrading)：

场景：目标 18 小时，断到 16 小时饿得不行了，改成 16:8。

逻辑：如果当前时长已超过新目标（例如已断 17h，改成 16h），圆环瞬间爆满，触发“完成撒花”动画。

文案：“您已超额完成任务！计划已调整。”
